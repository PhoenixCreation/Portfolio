Old Portfolio

You might be thinking that who puts old portfolio as a project? but this is not just a showcase portfolio. It is also a great example of full-stack website.

Description:

This website is perfect example of how to showcase things and also the database management. It has a todo section which is accessible through auth verfication system.
It also showcases CRUD operations on database.
It also has a Diary section which can easily be converted to the blog section beacuse it was built for blogging in mind and ended up in something else.
If you want to have a look go here.

Technology stack:

* HTML/CSS
* JavaScript
* Django
* Postgresql

Features:

* Simple yet eligant design.
* Mobile freindly.
* Simple animations with page loader.
* Direct mail contact capabilities through SMTP.
* Auth system with conditional rendering.
* SEO friendly.
* Blog ready.
* And many more...
